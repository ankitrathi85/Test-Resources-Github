We help developers and QA engineers discover high-quality test automation tools and frameworks. Our platform automatically scans GitHub repositories, analyzes their quality, and presents them in an easy-to-browse format.

üìä Quality Scoring

Each repository is scored on a 100-point scale across six categories:

  • Popularity (25 pts): GitHub stars and forks
  • Activity (20 pts): Recent commits and releases
  • Documentation (20 pts): README quality and examples
  • Community (15 pts): License and contributing guidelines
  • Maintenance (10 pts): Regular releases
  • Code Quality (10 pts): Topics, CI/CD, archived status

üèÜ Grading Scale

A+ 90-100 points - Exceptional quality
A 80-89 points - High quality
B 70-79 points - Good quality
C 60-69 points - Average quality
D 50-59 points - Below average
F 0-49 points - Poor quality

üîÑ Update Process

Our system automatically scans GitHub every 3 days using GitHub Actions. We respect API rate limits and use smart caching to minimize requests while keeping data fresh.

üìù Data Sources

All data comes from the GitHub public API. We analyze repository metadata, commit history, releases, documentation, and community health metrics.
